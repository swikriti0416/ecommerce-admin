E-Commerce Admin Dashboard
A simple and modern admin panel for managing an e-commerce store.
Built with React, Vite, and Tailwind CSS.
Features

Orders management (list, search, filter by status, delete)
Responsive and clean UI using Shadcn/UI components
Mock data with localStorage persistence (for demo)

Tech Stack

React
Vite
Tailwind CSS
Shadcn/UI
Lucide React (icons)
React Router DOM
React Toastify

Installation
git clone https://github.com/swikriti0416/ecommerce-admin.git
cd ecommerce-admin
npm install
npm run dev
